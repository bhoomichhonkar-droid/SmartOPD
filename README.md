# SmartOPD 

SmartOPD is a modern online OPD (Outpatient Department) appointment and queue management system built using Next.js. The platform helps patients book appointments digitally while enabling hospitals and clinics to manage appointments, doctor schedules, and queues efficiently.

---

## Features

*  Online appointment booking
*  ai symptomps detector
*  emergency button 
* Login and authentication system
* Admin dashboard
* Doctor and admin can see how many appointments there
*  Queue management system
*  Responsive user interface
*  Fast performance with Next.js

---

## Tech Stack

### Frontend

* Next.js
* React.js
* Tailwind Css
* Git & GitHub
* VS Code
* Local API handling using Next.js App Router

##  Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/bhoomichhonkar-droid/SmartOPD.git
```

###  Open Project Folder

```bash
cd SmartOPD
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Open in Browser

```bash
http://localhost:3000
```

## Problem Statement

Hospitals and clinics often face issues such as long waiting times, manual appointment handling, and inefficient queue management. SmartOPD digitizes the OPD workflow to improve patient experience and operational efficiency.
